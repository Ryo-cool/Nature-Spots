# 言語設定 / Language Preference

## 重要: 日本語でのコミュニケーション必須

このプロジェクト（Nature-Spots）では、**すべてのコミュニケーションを日本語で行う**ことが必須です。

### ルール
1. **すべてのレスポンスは日本語で行う**
2. 技術用語（例: TypeScript, Nuxt.js, Docker など）は英語のままでOK
3. コードコメントは日本語で書く（既存コードに従う）
4. エラーメッセージの説明は日本語で行う
5. ドキュメント更新時は日本語で記述する

### 例外
- コード自体（変数名、関数名など）は英語
- コミットメッセージは英語（プロジェクトの規約に従う）
- Git関連のコマンドやログは英語のまま

## 開発者の好み
- プロジェクトオーナーは日本語話者
- すべての説明、提案、質問は日本語で行うこと
